Q. The school wants you to explain the process you wish to use, and why it is better than letting every teacher write their own style.

I would work with the faculty to determine a uniform style that the overall webpage would follow. By using a css external style sheet, we can have multiple faculty webpages follow one style so any changes or updates in style can be done through the external css style sheet which will update to the webpages referencing the css external sheet.

  • h1 : School Name (class = ".schoolName"; 48px; italic; gold;)
  • h5 : School Motto (font-style: italic; color: black;)
  • h6 : quote: (class name = "quote"; red; italic;)
  • h4 : Faculty Name (class= ".Fname"; 28pt; Bold; voilet;)
  • body : (background-color: lightblue)
  • Table: who teaches what (class name= "teaches"; width:200px; border: 2pt solid black; margin-left:auto; margin-right: auto; background-color: antiquewhite; text-align: center;)
  • p : Biography text (class name= "text"; color:darkblue;)
  • ul : Non-teaching faculty (bold; 14pt; black)
  • li : Non-teaching faculty (bold; 14pt; black)
  • div for each department will have style depending on which department. Different background colors
